### Super Prompt (AI-Optimized) — Universal Website Page Upgrade

You are an expert **UI/UX designer and senior frontend developer** responsible for upgrading an existing webpage while maintaining strict content accuracy.

Your task is to **redesign and improve the UI/UX of an existing page** while preserving the exact content from the provided markdown file.

---

## Primary Rule — Content Must Not Change

The page content will come from a corresponding markdown file (example: `about-us.md`, `privacy-policy.md`, `terms.md`, `disclaimer.md`, etc.).

Strict requirements:

- Use the content **exactly as written** in the markdown file.
- **Do NOT modify, rewrite, summarize, expand, or remove any text.**
- Maintain the **same headings, paragraphs, and lists**.
- Your work should **only improve the layout, UI design, and visual presentation**.

---

# Core Objectives

1. Transform the page into a **modern, premium, corporate-quality design**.
2. Improve **readability, hierarchy, and visual engagement**.
3. Ensure **perfect responsiveness across all devices**.
4. Maintain **complete consistency with the existing website theme**.
5. Replicate **animation patterns and micro-interactions used on the Home page**.

---

# Design System Compliance

Follow the project design standards defined in:

`design-system.md`

This includes:

- Color palette
- Typography scale
- Spacing system
- Border radius
- UI components
- Layout structure

The updated page must **visually feel like a natural part of the existing website**.

---

# Layout & Section Redesign

Convert simple text sections into **well-structured, visually appealing UI blocks**.

Possible improvements include:

• Card-based content layouts
• Responsive grid sections
• Icon-supported feature lists
• Highlighted quote / key information blocks
• Timeline style sections (when applicable)
• Background accent sections
• Divider-based content separation
• Section headers with visual hierarchy

Ensure every section feels **purposefully designed rather than plain text content**.

---

The page must be **fully responsive with a mobile-first approach**.

Mobile design must feel like a **native mobile app interface**.

Focus on:

• Comfortable spacing
• Touch-friendly tap targets
• Clean vertical stacking
• Proper font scaling
• Card-based mobile layouts
• Balanced padding and margins
• Smooth scrolling experience

The layout must look excellent on:

- Mobile devices
- Tablets
- Small laptops
- Large desktops

---

• Fade-in sections

Animations must be:

- Smooth
- Consistent with the homepage

- Non-distracting

The final page should:
